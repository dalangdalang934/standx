# StandX Maker Bot - Windows 使用说明

## 方式一：直接使用（推荐）

### 前置要求
- 安装 Python 3.10 或更高版本
- 下载地址: https://www.python.org/downloads/
- 安装时勾选 "Add Python to PATH"

### 使用步骤

1. **双击运行 `start.bat`**
   - 首次运行会自动安装依赖（需要 1-2 分钟）
   - 浏览器会自动打开控制面板

2. **配置参数**
   - 输入私钥
   - 选择预设模式或手动调整参数
   - 点击"启动"按钮

3. **停止运行**
   - 点击"停止"按钮
   - 或直接关闭命令行窗口

---

## 方式二：打包成 EXE

### 打包步骤

在 Windows 上运行：

```bash
python build_exe.py
```

打包完成后，`dist/StandXMakerBot.exe` 就是独立的可执行文件。

### 使用 EXE

1. 双击 `StandXMakerBot.exe`
2. 浏览器自动打开控制面板
3. 配置并启动

---

## 注意事项

1. **私钥安全**
   - 私钥仅在本地使用，不会上传到任何服务器
   - 建议使用单独的交易账户

2. **网络要求**
   - 需要能够访问 StandX API
   - 需要稳定的网络连接

3. **风险提示**
   - 加密货币交易存在高风险
   - 请使用自己能承受损失的资金
   - 建议先在测试环境熟悉操作

---

## 文件说明

| 文件 | 说明 |
|------|------|
| `start.bat` | 一键启动脚本（推荐） |
| `build_exe.py` | 打包成 EXE 的脚本 |
| `web_ui.py` | Web UI 主程序 |
| `main.py` | 命令行模式运行 |
| `config.yaml` | 配置文件（首次运行后生成） |
